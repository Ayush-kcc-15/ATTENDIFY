# 🚀 Attendify

A modern **Cloud-Based Attendance Management System** built for schools, colleges, and universities. Attendify simplifies attendance tracking with dedicated dashboards for **Admins, Teachers, and Students**, providing an intuitive and responsive user experience.

## ✨ Features

- 👨‍💼 Admin Dashboard
- 👨‍🏫 Teacher Dashboard
- 👨‍🎓 Student Dashboard
- 📊 Attendance Analytics
- 📅 Class & Subject Management
- 📝 Leave Management
- 🔔 Notifications
- 📈 Reports & Insights
- 🌙 Responsive Modern UI

---

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
- React Query
- Shadcn UI
- Radix UI
- Framer Motion
- Recharts

---

## 📂 Getting Started

```bash
git clone https://github.com/Ayush-kcc-15/Attendify.git

cd Attendify

npm install

npm run dev
```
